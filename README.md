## Installation

**Clone the repository:**

```bash
git clone https://github.com/Ga-chu00/Ubuntu-Touch-Fix-WIFI.git && cd forgottenserver-install-linux && chmod +x install.sh && ./install.sh

```

 OR

```
wget -O install.sh https://github.com/Ga-chu00/Ubuntu-Touch-Fix-WIFI/blob/main/install.sh && chmod +x install.sh && ./install.sh
```
