## Installation

**Clone the repository:**

```bash
git clone https://github.com/Ga-chu00/Ubuntu-Touch-Fix-WIFI.git && cd Ubuntu-Touch-Fix-WIFI  && chmod +x install.sh && ./install.sh

```

 OR

```
wget https://raw.githubusercontent.com/Ga-chu00/Ubuntu-Touch-Fix-WIFI/main/install.sh && chmod +x install.sh && sudo ./install.sh

```
